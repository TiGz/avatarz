# Avatarz

AI-powered avatar generation app that transforms your photos into stunning stylized avatars using Google's Gemini API.

**Live Demo:** [avatarz.tigz.me](https://avatarz.tigz.me)

## Features

- Multiple avatar styles across categories (animated, artistic, professional, fantasy, etc.)
- Custom prompt support for unlimited creativity
- Name overlay options (graffiti, gold chain, headband, jersey, etc.)
- Photo library for storing and reusing input photos
- Gallery to view and manage generated avatars
- Daily generation limits with admin controls
- Invite-only access with magic link authentication

## Tech Stack

- **Frontend:** React 19, TypeScript, Vite, Tailwind CSS, Framer Motion
- **UI:** shadcn/ui, Lucide icons
- **Backend:** Supabase (Postgres, Auth, Storage, Edge Functions)
- **AI:** Google Gemini 3 Pro Image Preview
- **Hosting:** GitHub Pages

## Development

```bash
npm install
npm run dev
```
